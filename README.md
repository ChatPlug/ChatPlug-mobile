# 🏃 Running application

### What do you need?

- [Node.js](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/)

### 🤘 Let's start

- Clone repo</br>
   - Via https
   
    ```
    git clone https://github.com/ChatPlug/ChatPlug-mobile.git
    ```
   - Via ssh

    ```
    git clone git@github.com:ChatPlug/ChatPlug-mobile.git
    ```

- Now enter to app folder
  ```
  cd ChatPlug-mobile
  ```
- Install depedencies
  ```
  npm i
  ```
- Running application
  ```
  npm start
  ```
- Testing application
  ```
  npm test
  ```
